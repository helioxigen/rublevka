### Установка пакетов

```
npm install
```

### Локальный сервер

```
APP=%module% npm run start:dev
```

APP — это параметр, какое приложение фронтэнда запустить. Например, `cem` или `jqestate`. Все приложения смотреть в `src/core/config/apps.js`.

### Билд

```
APP=%module% npm run build
```

### Серверный рендеринг

```
APP=%module% npm run build # сначала билд статики
APP=%module% npm run start # потом запуск
```
